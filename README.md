# Automated_data_cleaning_tool
Automates data cleaning tasks like handling missing values, duplicates, and formatting issues.
Automated Data Cleaning Tool is a Python-based utility that streamlines the data preprocessing pipeline. It automatically handles common data quality issues such as:

Missing or null values, 
Duplicate rows, 
Inconsistent date/text formatting, 
Wrong data types, 
Outlier detection

Simply input a raw CSV or Excel file and get back a clean, analysis-ready dataset.
